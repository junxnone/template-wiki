# 自动生成 Sidebar


- 自动生成 `markdown` 格式的 sidebar
- 格式以空格为间隔分割 `Label` 
  - Issue Name: `L1 L2 L3 ...`
  - Markdown File: `L1_L2_L3_(...).md`

生成 sidebar格式
```
- L1
  - L2
    - L3
```
